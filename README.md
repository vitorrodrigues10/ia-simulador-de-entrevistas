# ia-simulador-de-entrevistas
Prompt para simular uma entrevista para uma carreira desejada.

Projeto criado para auxiliar no aprendizado de Prompt Engineering, em um Bootcamp da DIO. Ele tem a finalidade de auxiliar candidatos e entrevistadores para simular uma entrevista em um nicho específico, mas com foco em tecnologia.

O prompt utilizado está em um arquivo no repositório, o IA generativa escolhida foi o Google Gemini Pro.

O link para a conversa de teste do prompt é: https://gemini.google.com/share/f8545f46caa9
